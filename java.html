✅ Java Programming Roadmap (Beginner to Pro)

🔹 1. Introduction to Java

  • Platform-independent (WORA: Write Once, Run Anywhere)
  • Object-Oriented
  • Used for Android apps, web apps, backend systems, etc.

🔹 2. Basic Structure of Java Program

        public class Hello {
           public static void main(String[] args) {
            System.out.println("Hello, Neha!");
    }
}
  • class: Blueprint
  • main(): Entry point
  • System.out.println(): Print output

🔹 3. Variables and Data Types

  • primitive: int, float, char, boolean, double
  • non-primitive: String, Array, Class
        int age = 19;
        String name = "Neha";

🔹 4. Operators

  • arithmetic: + - * / %
  • relational: == != > < >= <=
  • logical: && || !
  • assignment: = += -=

🔹 5. Conditional Statements

  • if, else, else if
  • switch

🔹 6. Loops

  • for, while, do-while
    for (int i = 1; i <= 5; i++) {
        System.out.println(i);
}

🔹 7. Arrays

    int[] marks = {90, 80, 70};
    System.out.println(marks[1]); // Output: 80

🔹 8. Strings

    String name = "Neha";
    System.out.println(name.length());
    System.out.println(name.toUpperCase());

🔹 9. Object-Oriented Programming (OOP) – Core of Java

Key OOP Concepts:

  1. Class and Object
  2. Encapsulation
  3. Abstraction
  4. Inheritance
  5. Polymorphism

Class & Object Example:

    class Student {
      int age;
      String name;

      void display() {
        System.out.println(name + " is " + age + " years old.");
    }
}

public class Main {
    public static void main(String[] args) {
        Student s = new Student();
        s.name = "Neha";
        s.age = 19;
        s.display();
    }
}

🔹 10. Methods (Functions)

  • Method Overloading (Compile-time Polymorphism)
  • Method Overriding (Run-time Polymorphism)
    int add(int a, int b) {
        return a + b;
}

🔹 11. Inheritance

class Animal {
    void sound() {
        System.out.println("Animal makes sound");
    }
}

class Dog extends Animal {
    void bark() {
        System.out.println("Dog barks");
    }
}

🔹 12. Interfaces and Abstract Classes

  • Interface = 100% abstract methods
  • Abstract class = mix of abstract + non-abstract methods

🔹 13. Exception Handling

    try {
    int a = 10 / 0;
} catch (ArithmeticException e) {
    System.out.println("Cannot divide by zero");
}

🔹 14. File Handling

    import java.io.*;

public class FileDemo {
    public static void main(String[] args) {
        try {
            FileWriter fw = new FileWriter("file.txt");
            fw.write("Hello Neha!");
            fw.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
